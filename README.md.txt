# Web Scraper - Job Listings

This Python project scrapes job listings from a sample website and exports the data into a CSV file.

## Features
- Scrapes job title, company name, and location
- Exports data to CSV (`jobs.csv`)
- Easy to run with Python 3.14+
- Uses `requests`, `BeautifulSoup`, and `pandas` libraries

## Installation
1. Install Python 3.14 or higher
2. Install required libraries:
```bash
pip install requests beautifulsoup4 pandas
